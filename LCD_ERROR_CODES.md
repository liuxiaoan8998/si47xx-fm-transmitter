# Si47xx FM Transmitter - LCD 错误码系统

## 简化版错误码（2 位 LCD 屏）

### 速查表

| 错误码 | 名称 | 描述 | 解决方法 |
|--------|------|------|----------|
| **10** | I2C-W | I2C 写失败 | 检查 SDA/SCL 连接 |
| **11** | I2C-R | I2C 读失败 | 检查上拉电阻 |
| **12** | I2C-T | I2C 超时 | 检查芯片供电 |
| **20** | CHIP-I | 初始化失败 | 检查复位时序 |
| **21** | CHIP-D | 芯片检测失败 | 确认芯片型号 |
| **30** | F-RNG | 频率范围错误 | 调整到 87.5-108MHz |
| **31** | F-SET | 频率设置失败 | 重试或重启 |
| **40** | P-RNG | 功率范围错误 | 功率 77-115 dBuV |
| **41** | P-SET | 功率设置失败 | 检查芯片状态 |
| **99** | OK | 测试通过 | 设备正常 |

### 使用示例
```c
// 错误处理
if (si47xx_hw_write(...) != 0) {
    LCD_dispnum(1);  // 显示十位
    LCD_dispnum(0);  // 显示个位 (错误码 10)
    return 10;
}

// 测试通过
LCD_dispnum(9);  // 显示十位
LCD_dispnum(9);  // 显示个位 (99 = OK)
```

| **301** | F-RNG | 频率超范围 | 87.5-108MHz |
| **302** | F-SET | 频率设置失败 | 重试 |

## 接口函数

两个版本都需要实现：

```c
// 显示单个数字 (0-9)
extern void LCD_dispnum(char number);

// 毫秒延时
extern void si47xx_delay_ms(uint32_t ms);
```

## 显示时序

```c
// 显示 2 位错误码 (例如：10)
LCD_dispnum(1);        // 十位
si47xx_delay_ms(100);  // 延时稳定
LCD_dispnum(0);        // 个位
si47xx_delay_ms(100);

// 显示 3 位错误码 (例如：101)
LCD_dispnum(1);        // 百位
si47xx_delay_ms(50);
LCD_dispnum(0);        // 十位
si47xx_delay_ms(50);
LCD_dispnum(1);        // 个位
si47xx_delay_ms(50);
```
