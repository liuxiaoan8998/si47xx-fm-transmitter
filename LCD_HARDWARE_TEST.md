# Si47xx 硬件测试程序

## 简化版测试（推荐）

使用 2 位 LCD 屏显示错误码（10-99），代码简洁，适合快速测试。

### 文件
- `lcd_simple_test.c` - 简化版测试程序

### 编译运行
```bash
make lcd_simple
./lcd_simple_test
```

### 错误码（10-99）

| 错误码 | 名称 | 含义 | 解决方法 |
|--------|------|------|----------|
| **10** | I2C-W | I2C 写失败 | 检查 SDA/SCL 连接 |
| **11** | I2C-R | I2C 读失败 | 检查上拉电阻 |
| **12** | I2C-T | I2C 超时 | 检查芯片供电 |
| **20** | CHIP-I | 初始化失败 | 检查复位时序 |
| **21** | CHIP-D | 芯片检测失败 | 确认芯片型号 |
| **30** | F-RNG | 频率范围错误 | 检查频率设置 (87.5-108MHz) |
| **31** | F-SET | 频率设置失败 | 重试或重启 |
| **40** | P-RNG | 功率范围错误 | 功率应在 77-115 dBuV |
| **41** | P-SET | 功率设置失败 | 检查芯片状态 |
| **99** | OK | 测试通过 | 设备正常 |

### 接口要求
```c
extern void LCD_dispnum(char number);  // 显示数字 0-9
extern void si47xx_delay_ms(uint32_t ms);  // 延时函数
```

## 完整版测试（3 位 LCD 屏）

使用 3 位 LCD 屏显示详细错误码（101-999），适合深入调试。

### 文件
- `lcd_test_si47xx.c` - 完整版测试程序
- `test_with_lcd.c` - 集成测试

### 编译运行
```bash
make lcd_test
./test_with_lcd
```

### 错误码分类（101-999）

| 范围 | 类型 |
|------|------|
| 101-199 | I2C 通信错误 |
| 201-299 | 芯片初始化错误 |
| 301-399 | 频率设置错误 |
| 401-499 | 功率设置错误 |
| 501-599 | RDS 功能错误 |
| 601-699 | 音频控制错误 |
| 701-799 | 属性设置错误 |
| 801-899 | 配置错误 |
| 901-999 | 系统错误 |

详细错误码参考 `LCD_ERROR_CODES.md`。
